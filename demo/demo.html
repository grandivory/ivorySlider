How to Use Options Public Methods Extending Transitions

How to use:

  1. Add ivorySlider.js and ivorySlider.transitions.js to your page
  2. Optionally, add in the default styles in ivorySlider.css
  3. Create html markup for each slide within a container (usually <ul><li>)
  4. After the page has loaded, call .ivorySlider() on your container

Public methods

These methods can be called by passing a string as the first argument to .ivorySlider() after it has already been initialized on an element. Any further arguments are passed to the function. If the function is called on a single slider, that functions return value will be passed back.

$('#slides').ivorySlider('slideTo', 3);
Method Name Description
next Continue to the next slide
prev Move to the previous slide
slideTo Move the slider to a particular slide
Arguments: index
options Overwrite existing options on the slider
Arguments: Object
play Set the slider to automatically transition between slides. Advance one slide if no timer is visible.
pause Stop the slider from automatically transitioning between slides.

Extending Transitions

One of the best features of ivorySlider is the ability for end-users to add their own transition effects to their slides. Doing so is incredibly simple. Just add a new function to jQuery.ivorySlider.effects with the following template:

function(options, fromSlide, toSlide)

The options argument contains all of the options passed in to the slide when it was instantiated, plus any defaults for options that weren't specified. The fromSlide and toSlide arguments contain the jQuery objects for the slides themselves. Within the function, this refers to the slider object, with the properties lastSlide and currentSlide set to the indices of the "from" and "to" slides, respectively. Once any animations are complete, call slider.finishedTransition to complete clean-up for the slider and trigger auto-play if required.

As an example, the following is the code for the simplest transition packed with ivorySlider - the fade transition. This code will fade the current slide out, then fade the next slide in.

(function($) {
  $.ivorySlider.effects.fade = function(vars, fromSlide, toSlide) {
    slider = this;
    fromSlide.fadeOut({
      duration: vars.speed,
      easing: vars.easing,
      complete: function() {
        toSlide.fadeIn({
          duration: vars.speed,
          easing: vars.easing,
          complete: function() {
            slider.finishedTransition();
          }
        });
      }
    });
  };
}(jQuery));
